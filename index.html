Opticalc est une application sûre. Si vous voyez un avertissement comme "Ce fichier n'est pas souvent téléchargé" ou "Windows a protégé votre PC", voici quoi faire :
  • Sur Google Chrome : Cliquez sur la flèche à côté de l'avertissement, puis choisissez « Conserver » ou « Conserver quand même ».
  • Sur Microsoft Edge : Cliquez sur « Conserver » → « Afficher plus » → « Conserver quand même ».
  • Sur Windows (SmartScreen) : Cliquez sur « Informations complémentaires » puis « Exécuter quand même ».

Nous travaillons à signer numériquement l'application pour éviter ces avertissements à l'avenir.
